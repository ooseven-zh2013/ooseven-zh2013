# 2026-06-14 工作日志

## CI/CD: 配置 lowlighter/metrics 每日自动生成 GitHub Stats

- 创建 `.github/workflows/metrics.yml`
  - 每天 UTC 00:00（北京时间 08:00）自动运行
  - 支持 `workflow_dispatch` 手动触发
  - 生成 4 个 SVG：主要统计、等距贡献日历、语言统计、AniList
- 更新 `README.md`，嵌入生成的 SVG 统计图
- **需要用户操作**：在 GitHub 仓库 Settings → Secrets → Actions 中创建 `METRICS_TOKEN`，值为具有 `repo` 和 `user` 权限的 Personal Access Token (classic)
